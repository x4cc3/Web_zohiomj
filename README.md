# Flexitify

A modern music streaming web application built with vanilla JavaScript, HTML, and CSS.

## Features

- Music player with play/pause, next/previous controls
- Search functionality for artists, albums, and tracks
- Responsive design for mobile and desktop
- Artist profiles with track listings
- Library organization
- Smooth animations and modern UI

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome Icons

## Getting Started

1. Open index.html in a modern web browser
2. Navigate through different sections using the sidebar
3. Search for music, play tracks, and explore artist profiles
